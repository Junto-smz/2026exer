# プログラミング演習２ 課題リポジトリ

## 概要
2回生春学期の「プログラミング演習２」で作成した課題集です。Python を使って、連結リスト、各種ソート、スタック、キュー、グラフ、ヒープ、再帰などの基礎アルゴリズムとデータ構造の実装方法を学ぶことを目的としています。

## 収録内容
- 連結リスト実装
- スタック / キュー
- 各種ソートアルゴリズム
- グラフ探索
- ヒープ
- 再帰的な問題
- 期末試験・実践問題

## ディレクトリ構成
- `1-n/`
  - `1-1.py`～`1-5.py`
- `2-n/`
  - `2-1.py`～`2-4.py`
- `3-n/`
  - `3-1.py`～`3-5.py`
- `exam1-student/`
  - `exam1.py`, `exam2.py`, `exam3.py`, `test.py`
  - `def_listqueue.py`, `def_node.py`
- ルートディレクトリ
  - `4-2.py`, `4-3.py`, `4-4.py`, `4-5.py`
  - `5_1.py`, `5_2.py`, `5-3.py`
  - `7-1.py`, `7-2.py`, `7-3.py`
  - `week10_1.py`, `week10_2.py`, `week10_3.py`
  - `def_db.py`, `def_graph.py`, `def_heap.py`, `def_node.py`, `def_queue_stack.py`, `def_stop.py`, `fibo.py`

## 実行方法
各課題ファイルは Python で実行できます。例:

```bash
python 7-2.py
```

必要に応じて、問題ごとにファイル名を指定して実行してください。

## 学習した内容
- 単方向・双方向連結リスト
- スタック / キューの基本操作
- バブルソート、挿入ソート、選択ソートなどの比較
- グラフ探索（幅優先探索 / 深さ優先探索）
- ヒープ構造と優先度付きキュー
- 再帰関数とフィボナッチ数列

## 注意
- 各ファイルは授業課題の演習用コードです。
- 実行には Python 3.x が必要です。
